History

The history of this company happened when a group of friends got together to go rafting, we realized the need to be able to speed up the selection of places and the quality of those same places to be able to practice this sport, so we decided to create a simple way to be able to present all kinds of places where you could practice this and in a way that fits your money, so we were able to create Supido, to help both beginners and experts in being able to enjoy doing this with just one click.
